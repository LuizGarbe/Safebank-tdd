# Safebank-tdd
Sistema de Gestão de Folha de Pagamento com foco em TDD e Qualidade de Código. Desenvolvido em Python, o projeto implementa regras de negócio para gerenciamento de funcionários, utilizando Pytest para testes unitários e Coverage para garantir a cobertura e integridade das funções de cálculo salarial, bônus e validações societárias.
